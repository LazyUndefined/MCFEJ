## Introduction
    - MCFEJ は GUI の mcfunction エディタです。
    - 動作には Common Lisp 処理系 (SBCL を推奨)、Quicklisp 、Tcl/Tk が必要です。
    - 開発は超スローペースになりますが、どうぞよろしくお願いします。
